# WebOS

WebOS is a browser-based operating system that recreates a desktop experience inside a web page. It includes a growing collection of built-in applications and serves as a place to showcase my projects while experimenting with web development.

## Features

### Projects

View projects that I've created, along with descriptions and links to learn more about each one.

### Calculator

A simple calculator for everyday calculations.

### Browser

A built-in browser that allows you to visit websites and search the web without leaving WebOS.

### Notes

A simple notes app for creating and saving text notes inside WebOS.

### Music

A music player app that lets you play songs inside WebOS. It includes "Life Goes On" by Oliver Tree as the featured song.

### Settings

A settings app for managing WebOS options and customization.

### Weather

A weather app that displays weather information inside the WebOS environment.
### Interface

* Desktop-style layout
* Windowed applications
* Robot-inspired design
* Lightweight and easy to use

## Why I Built It

I wanted to see how much of a desktop operating system I could recreate using only web technologies. The project started as an experiment and has gradually grown as I added more applications and improved the interface.
